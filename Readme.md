# HTML5_Informatica

**Computer Scienze Course Webpages**

---

Questo repository contiene esercizi in HTML5.
Le pagine includono esempi pratici e progetti web che coprono diversi argomenti del corso.

---

## Struttura della repository
Ogni cartella è un progetto/esercizio che viene svolto in classe

|HTML5_Informatica
|
|- Esercizio 1
|- Esercizio 2
|- Progetto 1
|- Progetto 2
|- ...

---

## Contenuti
Il progetto include:
- Pagine `HTML5`
- Alcuni temi `CSS3` ognuno per il proprio progetto
- Script `JavaScript` per l'interattività con la pagina `HTML`
- Files `Json` o `CSV` che vengono utilizzati per gestire dati
